# EP Portal Employee Assessment Questions

## Overview
These questions are designed to test employee understanding of the EP Portal system features and functionality. Employees should be able to answer these questions after using the system.

---

## Assessment Questions

### 1. **Dashboard Navigation**
**Question:** What are the 8 main sections available in the employee sidebar navigation menu?

**Expected Answer:** 
- Dashboard
- My Profile  
- Leave Requests
- My Tasks
- Support Tickets
- Salary Withdrawal
- Documents
- Reminders

---

### 2. **Profile Management**
**Question:** Which profile fields can employees edit themselves, and which fields are read-only? Name at least 3 editable and 3 read-only fields.

**Expected Answer:**
**Editable fields:** First Name, Last Name, Middle Name, Date of Birth, Gender, Phone Number, Address, Profile Picture
**Read-only fields:** Email, Employee Number, Department, Position, Salary

---

### 3. **Leave Request Process**
**Question:** What are the required steps and information needed to submit a leave request? What happens after submission?

**Expected Answer:**
- Select leave type from dropdown
- Choose start and end dates (cannot be in the past)
- Provide detailed reason
- System automatically calculates working days (excludes weekends)
- After submission: Admin receives email/SMS notification, employee gets SMS confirmation
- Employee can view status (pending/approved/rejected) and track progress

---

### 4. **Task Management**
**Question:** What are the 4 possible task statuses, and what actions can an employee take on tasks in different statuses?

**Expected Answer:**
**Task Statuses:** Pending, In Progress, Completed, Cancelled
**Actions:**
- Pending tasks: Can start (change to In Progress), view details, update progress
- In Progress tasks: Can mark complete, update progress/notes, view details
- Completed/Cancelled tasks: Can only view details (no actions available)

---

### 5. **Support Ticket System**
**Question:** What information is required when creating a support ticket, and what file types can be attached?

**Expected Answer:**
**Required information:** Subject, Description, Category, Priority
**Optional:** File attachments
**Allowed file types:** PDF, DOC, DOCX, JPG, JPEG, PNG, GIF, TXT, XLSX, XLS
**File size limit:** 5MB per file
**Process:** System generates unique ticket number, admin receives notification, employee gets confirmation

---

### 6. **Salary Withdrawal Process**
**Question:** What are the two payment methods available for salary withdrawal, and what are the minimum withdrawal amounts for each?

**Expected Answer:**
**Payment Methods:**
1. **Bank Transfer** - Minimum withdrawal varies by bank
2. **Mobile Money** - Minimum withdrawal varies by provider (MTN, Airtel, etc.)
**Process:** Employee requests withdrawal → System calculates charges → Deducts from available balance → Processes via Flutterwave → Employee receives confirmation
**Note:** Charges are automatically deducted from withdrawal amount

---

### 7. **Document Management**
**Question:** What types of documents can employees upload, what is the maximum file size, and who can delete uploaded documents?

**Expected Answer:**
**Allowed file types:** PDF, DOC, DOCX, JPG, JPEG, PNG
**Maximum file size:** 10MB
**Upload process:** Employees can upload their own documents with custom names
**Deletion rights:** Only admins can delete documents; employees can only view and download their own documents

---

### 8. **Reminder System**
**Question:** What delivery methods are available for reminders, and what information is required to create a reminder?

**Expected Answer:**
**Delivery Methods:** SMS, WhatsApp, Email, System notifications
**Required Information:**
- Title (required)
- Reminder date and time (required)
- Description (optional)
- Priority level
- Delivery method selection
**Process:** System automatically sends reminders at scheduled time via selected method(s)

---

### 9. **Notification System**
**Question:** How do employees receive notifications in the EP Portal, and what actions can they take on notifications?

**Expected Answer:**
**Notification Types:** System notifications appear in dashboard, plus external notifications via SMS/Email/WhatsApp
**Dashboard Features:**
- View unread notification count
- Click notifications to mark as read
- Mark all notifications as read at once
- Delete individual notifications
- Notifications show with "New" badge until read

---

### 10. **System Security & Login**
**Question:** What is the login process for employees, and what authentication methods are used?

**Expected Answer:**
**Login Process:**
1. Enter email and password on login page
2. System sends OTP (One-Time Password) via SMS, Email, or WhatsApp
3. Enter OTP on verification page to complete login
4. Access employee dashboard

**Security Features:**
- Multi-factor authentication (password + OTP)
- Secure session management
- Automatic logout after inactivity
- Role-based access (employees cannot access admin functions)

---

## Scoring Guide

**Excellent (90-100%):** Employee demonstrates comprehensive understanding of all system features and can explain processes in detail.

**Good (80-89%):** Employee understands most features with minor gaps in knowledge.

**Satisfactory (70-79%):** Employee has basic understanding but may need additional training on some features.

**Needs Improvement (Below 70%):** Employee requires comprehensive retraining on system usage.

---

## Additional Notes for Trainers

- These questions can be used as verbal assessments or written tests
- Consider practical demonstrations alongside theoretical questions
- Focus on areas where employees show weakness for additional training
- Update questions as new features are added to the system
- Encourage employees to explore all features hands-on before assessment
